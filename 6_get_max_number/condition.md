# get_max_number

## **Условие**

Напиши функцию **get_max_number**. Она принимает список чисел и возвращает максимальное из них. Встроенную функцию **max()** использовать нельзя.

## Пример 1

```
Входные данные:
[100, 200, 200, 1377]

Выходные данные:
1377
```

## Пример 2

```
Входные данные:
[1, 1, 1, 1, 1]

Выходные данные:
1
```
