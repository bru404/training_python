# get_sum_digits

### **Условие**

Напиши функцию **get_sum_digits**, которая принимает в качестве параметра целое число и складывает его цифры. Нужно вернуть результат сложения.

### Пример 1

```
Входные дынные:
1111

Выходные данные:
4
```

### Пример 2

```
Входные данные:
1984

Выходные данные:
22
```

### Пример 3

```
Входные данные:
000000

Выходные данные:
0
```
